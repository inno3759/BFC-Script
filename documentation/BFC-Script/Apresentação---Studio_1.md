O **BFCfc-Script** é um framework que possibilita a integração de scripts com aplicações. O framework oferece um ambiente de desenvolvimento, compilação e execução de scripts, bem como uma linguagem amigável para usuários não técnicos e APIs utilitárias.

A linguagem possui poucos comandos e algumas similaridades com a linguagem JavaTM, como os operadores (lógico, comparação, atribuição e aritmético), o uso de chaves para circundar blocos, sintaxe _case sensitive_ , etc. 

Para simplificar os _scripts_ , não é necessário usar ponto e vírgula no final de cada instrução e os comandos foram desenvolvidos em português, para simplificar o seu uso a usuários sem experiência em linguagens de programação.
